Output Documentation

Summary of Arriba's results concerning detection of gene fusions in transcriptome profiles.
Attention: This overview contains the results for all input samples (control & condition samples).

Column descriptions:

For more information on the columns, please see the Arriba documentation.

  1. sample_id: The analyzed sample.
  2. gene1: gene1 contains the gene which makes up the 5' end of the transcript
  3. gene2: gene2 contains the gene which makes up the 3' end of the transcript
  4. strand1.gene.fusion/strand2.gene.fusion: Two values seperated by a slash. The strand before the slash reflects the strand of the gene according to the gene annotation supplied to Arriba. The value after the slash reflects the strand that is transcribed. Occassionally, the strand that is transcribed cannot be predicted reliably. In this case, Arriba indicates the lack of information as a dot (.).
  5. breakpoint1: breakpoint1 contains the breakpoint of the 5' gene
  6. breakpoint2: breakpoint2 contains the breakpoint of the 3' gene
  7. site1/site2: These columns add information about the location of the breakpoints.
  8. type: Inferred type of the event.
  9. direction1/direction2: These columns indicate the orientation of the fusion.
  10. split_reads1/split_reads2: These columns contain the number of split reads that support the fusion.
  11. discordant_mates: Number of discordant mates that support the fusion.
  12. coverage1/coverage2: These two columns show the coverage near breakpoint1 and breakpoint2, respectively.
  13. confidence: Each prediction is assigned one of the confidences low, medium, or high.
  14. closest_genomic_breakpoint1/closest_genomic_breakpoint2: These two columns contain the coordinates of the genomic breakpoints which are closest to the transcriptomic breakpoints given in the columns breakpoint1 and breakpoint2.
  15. filters: This column lists the filters which removed one or more of the supporting reads.
  16. fusion_transcript: This column contains the assembled fusion transcript.
  17. reading_frame: This column states whether the gene at the 3' end of the fusion is fused in-frame or out-of-frame
  18. peptide_sequence: This column contains the fusion peptide sequence.
  19. read_identifiers: This column contains the names of the supporting reads separated by commas.
Method:

Arriba is a command-line tool for the detection of gene fusions from RNA-Seq data. It was developed for the use in a clinical research setting. Therefore, short runtimes and high sensitivity were important design criteria. It is based on the ultrafast STAR aligner, and the post-alignment runtime is typically just ~2 minutes.
