Output Documentation

The output is a summary of the "samtools stat" output for every single input sample.

Column descriptions:
  1. BAM.file: The name of the analyzed alignment file of a given sample.
  2. Total.reads Total number of included reads.
  3. Mapped.reads: Total number of included mapped reads.
  4. Forward strand: Total number of reads mapped against the forward strand.
  5. Reverse strand: Total number of reads mapped against the reverse strand.
  6. Failed QC: Total number of reads that failed Quality Control (flag 0x200 (512) is set).
  7. Duplicates: Number of duplicate reads (flag 0x400 (1024) is set).
  8. Paired.end.reads: Number of paired reads.
  9. X.Proper.pairs: Number of properly paired reads (flag 0x2 (2) is set).
  10. Both.pairs.mapped: Total number of pairs where both reads are mapped.
  11. Read.1: Number of first reads.
  12. Read.2: Number of second reads.
  13. Singletons: Number of reads without pairing.
Method:

Samtools Stats is part of the Samtools toolkit and produces comprehensive statistics from alignment files in the SAM/BAM/CRAM format. It is a useful tool for evaluating the quality of an alignment and can be used to compare the quality of different alignment methods.
