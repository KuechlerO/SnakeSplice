Output Documentation

Summary of the Private Junction Detection (PJD) analysis.
Attention: This summary displays only the top 250 detected private junctions, sorted by the number of reads supporting the junction.

Column descriptions:
  1. chrom: The chromosome of interest.
  2. exact_jct_start: The start position for the assessed junction.
  3. exact_jct_end: The end position for the assessed junction.
  4. strand: The strand of the assessed junction.
  5. add_info: Additional info concerning the genomic region/gene of the assessed junction.
  6. [sampleID]: Raw count number of reads that support the assessed junction in the given sample.
  7. total_sum: The total sum of reads that are supporting this junction.
  8. max_contrast: Predefined cutoff, which signifies how many reads/which percentage of reads would be allowed in the contrast group.
Method:

Private Junction Detection: The Private Junction Detection is a simple tool developed by Oliver Kuechler to detect splicing junctions that only occur (or specifically do not occur) in a single sample (or group of samples). The tool is based on the Regtools.
