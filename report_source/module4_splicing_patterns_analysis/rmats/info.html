Output Documentation

Summary of rMATS' result regarding the detection of differential alternative splicing events from RNA-Seq data.

Each alternative splicing event type has a corresponding set of output files: [SE (skipped exon), MXE (mutually exclusive exons), A3SS (alternative 3' splice site), A5SS (alternative 5' splice site), RI (retained intron)].
Additionally, [JC]-files include only reads that span junctions defined by rmats (Junction Counts), while [JCEC]-files include both reads that span junctions defined by rmats (Junction Counts) and reads that do not cross an exon boundary (Exon Counts).

Attention: Results have been prefiltered to only include significant events (p-value <0.05 and FDR < 0.10). If the results are more than 1000 lines, only the first 1000 lines are shown.

Column descriptions:

For event event-specific columns please visit the rMATS- ReadMe.

  1. ID: rMATS event id
  2. GeneID: Gene id
  3. geneSymbol: Gene name
  4. chr: Chromosome
  5. strand: Strand of the gene
  6. IJC_SAMPLE_1: Inclusion counts for sample 1. Replicates are comma separated
  7. SJC_SAMPLE_1: Skipping counts for sample 1. Replicates are comma separated
  8. IJC_SAMPLE_2: Inclusion counts for sample 2. Replicates are comma separated
  9. SJC_SAMPLE_2: Skipping counts for sample 2. Replicates are comma separated
  10. IncFormLen: Length of inclusion form, used for normalization
  11. SkipFormLen: Length of skipping form, used for normalization
  12. PValue: Significance of splicing difference between the two sample groups. (Only available if the statistical model is on)
  13. FDR: False Discovery Rate calculated from p-value. (Only available if statistical model is on)
  14. IncLevel1: Inclusion level for sample 1. Replicates are comma separated. Calculated from normalized counts
  15. IncLevel2: Inclusion level for sample 2. Replicates are comma separated. Calculated from normalized counts
  16. IncLevelDifference: average(IncLevel1) - average(IncLevel2)

Summary file: Brief summary of all AS event types. Includes the total event counts and significant event counts. By default, events are counted as significant if FDR <= 0.05.

Method:

rMATS: is a computational tool to detect differential alternative splicing events from RNA-Seq data. The statistical model of MATS calculates the P-value and false discovery rate that the difference in the isoform ratio of a gene between two conditions exceeds a given user-defined threshold. From the RNA-Seq data, MATS can automatically detect and analyze alternative splicing events corresponding to all major types of alternative splicing patterns. MATS handles replicate RNA-Seq data from both paired and unpaired study design.
