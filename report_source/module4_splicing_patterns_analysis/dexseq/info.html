Output Documentation

Summary of DEXSeq's results regarding the differential exon usage using RNA-seq exon counts between samples with different experimental designs.

Attention: Results have been prefiltered to only include significant events (p-value <0.05). If the results are more than 1000 lines, only the first 1000 lines are shown.

Column descriptions:

For event event-specific columns please visit the rMATS- ReadMe.

  1. ensembl_gene_id: Ensembl-ID of analyzed gene
  2. gene_name: Entrez gene name
  3. featureID: The considered features (i.e. exon)
  4. exonBaseMean: Mean of normalized counts for considered exon
  5. dispersion: Estimated exon dispersion (information on the strength of the noise is inferred from the biological replicates in the data set)
  6. stat: LRT statistic: Control vs Condition
  7. pvalue: Computed LRT p-value
  8. padj: Multiple testing corrected p-value (Benjamini-Hochberg).
  9. [Condition/Control]: Exon usage coefficient
  10. log2fold_[Control]_[Condition]: Relative exon usage fold change
  11. genomicData.seqnames: Chromosome name of feature location
  12. genomicData.start: Genomic start position of feature
  13. genomicData.end: Genomic end position of feature
  14. genomicData.width: Genomic width of feature
  15. genomicData.strand: Genomic strand location of feature
  16. transcripts: list of transcripts overlapping with the exon
Method:

DEXSeq is focused on finding differential exon usage using RNA-seq exon counts between samples with different experimental designs. It provides functions that allows the user to make the necessary statistical tests based on a model that uses the negative binomial distribution to estimate the variance between biological replicates and generalized linear models for testing. The package also provides functions for the visualization and exploration of the results.

The basic concept can be summarized as follows. For each exon (or part of an exon) and each sample, we count how many reads map to this exon and how many reads map to any of the other exons of the same gene. We consider the ratio of these two counts, and how it changes across conditions, to infer changes in the relative exon usage. In the case of an inner exon, a change in relative exon usage is typically due to a change in the rate with which this exon is spliced into transcripts (alternative splicing). Note, however, that DEU is a more general concept than alternative splicing, since it also includes changes in the usage of alternative transcript start sites and polyadenylation sites, which can cause differential usage of exons at the 5' and 3' boundary of transcripts.
