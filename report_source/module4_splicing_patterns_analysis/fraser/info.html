Output Documentation

Summary of FRASER's results: Detection of rare aberrant splicing events in transcriptome profiles. Significant events for aberrant splice junctions or splice sites, based on the specified cutoffs (default: adjusted p-value < 0.05), are displayed.

Column descriptions:
  1. sampleID: the sampleID in which this aberrant event occurred.
  2. hgncSymbol: the gene symbol of the gene that contains the splice junction or site if available.
  3. addHgncSymbols: Additional gene symbols of overlapping genes that contain the splice junction or site as well.
  4. type: the metric for which the aberrant event was detected (either psi5 for $\psi_5$, psi3 for $\psi_3$ or theta for $\theta$).
  5. pValue: the p-value of this event.
  6. padjust: the adjusted p-value of this event.
  7. zScore: the z-score of this event.
  8. psiValue: the value of $\psi_5$, $\psi_3$ or $\theta$ metric (depending on the type column) of this junction or splice site for the sample in which it is detected as aberrant.
  9. deltaPsi: the $\Delta\psi$-value of the event in this sample, which is the difference between the actual observed $\psi$ and the expected $\psi$.
  10. meanCounts: the mean count (k) of reads mapping to this splice junction or site over all samples.
  11. meanTotalCounts: the mean total count (n) of reads mapping to the same donor or acceptor site as this junction or site over all samples.
  12. counts: the count (k) of the splice junction or site for the sample where it is detected as aberrant.
  13. totalCounts: the total count (n) of the splice junction or site for the sample where it is detected as aberrant.
Method:

FRASER: Detection of rare aberrant splicing events in transcriptome profiles. Read count ratio expectations are modeled by an autoencoder to control for confounding factors in the data. Given these expectations, the ratios are assumed to follow a beta-binomial distribution with a junction specific dispersion. Outlier events are then identified as read-count ratios that deviate significantly from this distribution. FRASER is able to detect alternative splicing, but also intron retention. The package aims to support diagnostics in the field of rare diseases where RNA-seq is performed to identify aberrant splicing defects. .
