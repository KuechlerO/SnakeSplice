Output Documentation

Result of the analysis of differentially expressed genes. Quantification of the differentially expressed genes was done with Salmon, while the differential expression analysis was done with DESeq2. Only significant differentially expressed genes (adjusted p-value < 0.10) are shown. The experiment was done between the annotated control samples and one given condition (see workflow annotation).

Column descriptions:
  1. subject: Gene ID (Ensembl).
  2. baseMean: Normalized expression (TPM) mean of all samples in the dataset.
  3. log2FoldChange: Estimation: Log-2 Fold Change (Control vs. Condition).
  4. lfcSE: Standard error for the log-fold-change estimation.
  5. pvalue: Computed p-value (Wald test) for measuring this gene as differentially expressed.
  6. padj: Adjusted p-value (Benjamini Hochberg) for measuring this gene as differentially expressed.
  7. symbol: Official gene symbol (Entrez)
  8. entrez: Gene-ID (Entrez)
Method:

Salmon is a tool for quantifying the expression of transcripts using RNA-seq data. Salmon uses new algorithms (specifically, coupling the concept of quasi-mapping with a two-phase inference procedure) to provide accurate expression estimates very quickly (i.e. wicked-fast) and while using little memory. Salmon performs its inference using an expressive and realistic model of RNA-seq data that takes into account experimental attributes and biases commonly observed in real RNA-seq data.

DESeq2 estimates variance-mean dependences in count data from high-throughput sequencing assays and tests for differential expression based on a model using the negative binomial distribution.
