Output Documentation

Summary of OUTRIDER's result for the detection of aberrantly expressed genes by identying aberrant read counts in RNA-seq data. Only significant differentially expressed genes (adjusted p-value < 0.10) are shown.

Column descriptions:
  1. geneID: Gene ID (Ensembl) and its symbol are displayed.
  2. sampleID: The identified sample.
  3. pValue: The nominal P-value indicating the outlier status.
  4. padjust : The FDR corrected P-value indicating the outlier status.
  5. zScore: Computed z-score.
  6. l2fc: Computed log2-fold-change.
  7. rawcounts: The observed read counts.
  8. normcounts: The expected count given the fitted autoencoder model for the given gene-sample combination.
  9. meanCorrected: For this gene, the mean of the expected counts, respectively, given the fitted autoencoder model.
  10. theta: The dispersion parameter of the NB distribution for the given gene.
  11. aberrant: The outlier status of this event: Either TRUE or FALSE.
  12. AberrantBySample: Number of outliers for the given sample.
  13. AberrantByGene: Number of outliers for the given gene.
  14. padj_rank: Rank of this outlier event within the given sample.
Method:

OUTRIDER: Identification of aberrant gene expression in RNA-seq data. Read count expectations are modeled by an autoencoder to control for confounders in the data. Given these expectations, the RNA-seq read counts are assumed to follow a negative binomial distribution with a gene-specific dispersion. Outliers are then identified as read counts that significantly deviate from this distribution. Furthermore, OUTRIDER provides useful plotting functions to analyze and visualize the results.
